# Gopin Sora — Resume

A single-file, print-ready (PDF-style) resume. Open `index.html` in a browser and
use **Ctrl/Cmd + P → Save as PDF** to export.

- `index.html` — the resume (self-contained, embedded CSS)
- `profile.jpg` — header photo (add your own)
- `img/` — project thumbnails (see img/README.txt)
